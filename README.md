# Hi, I'm Arash

### Ecola Radio
A Python application that recreates the behavior of GTA III and Vice City radio stations, including continuous playback between station switches.

### ADF Audio Tool
Research and tooling for GTA Vice City radio audio files and formats.

### PassMan
A lightweight password manager written in Python.
