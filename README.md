# Hi, I'm Arash

I enjoy reverse engineering, game modding, and creating tools for classic games.

Most of my work focuses on understanding how games work internally, preserving technical knowledge, and building utilities that improve the player experience.

## Featured Projects

### GTA SA Save Patch Tool
Modify GTA San Andreas save files safely by editing censorship-related flags while automatically fixing checksums.

### Ecola Radio
A Python application that recreates the behavior of GTA III and Vice City radio stations, including continuous playback between station switches.

### ADF Audio Tool
Research and tooling for GTA Vice City radio audio files and formats.

### PassMan
A lightweight password manager written in Python.

### Game Collection
A collection of small games created for learning, experimentation, and fun.
